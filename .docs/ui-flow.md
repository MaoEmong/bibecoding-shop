# 화면/흐름 설계 (현재 구현 기준)

## 전역
- 공통 헤더: 로고, 상품 목록, 로그인/로그아웃, 장바구니(구매자), 마이페이지
- 로그인 상태와 역할에 따라 메뉴가 달라집니다.

## 공통 화면
### 홈
- 구성: 히어로 영역, 추천 상품 카드, 상품 미리보기
- 액션: 상품 목록으로 이동

### 상품 목록
- 구성: 상품 카드 리스트
- 액션: 상품 상세로 이동

### 상품 상세
- 구성: 상품명, 가격, 재고, 설명
- 액션: 장바구니 담기(구매자)

### 로그인
- 구성: 이메일/비밀번호
- 액션: 로그인, 회원가입 이동

### 회원가입
- 구성: 역할(판매자/구매자), 이메일/비밀번호/이름
- 액션: 회원가입 후 자동 로그인(홈으로 이동)

## 구매자 화면
### 장바구니
- 구성: 담긴 상품 목록, 수량 변경, 삭제, 합계 금액
- 액션: 주문하기

### 주문 페이지
- 구성: 주문 상품 요약, 총 금액
- 액션: 주문 확정

### 주문 완료
- 구성: 주문 번호, 주문 요약
- 액션: 주문 내역/상품 목록 이동

### 주문 내역
- 구성: 주문 리스트, 상태
- 액션: 주문 상세 이동

### 주문 상세
- 구성: 주문 번호, 상태, 생성 시각, 상품별 수량/금액

## 판매자 화면
### 판매자 상품 목록
- 구성: 본인 상품 리스트(가격/재고/상태)
- 액션: 등록, 수정, 삭제

### 상품 등록/수정
- 구성: 상품명, 설명, 가격, 재고
- 액션: 저장

### 판매자 주문 내역
- 구성: 주문 리스트(주문번호, 구매자, 상태, 금액)
- 액션: 상태 변경

## 다음 단계 화면
- 상품 검색/필터
- 주문 취소
- 배송 상세 추적
- 리뷰/문의/위시리스트
