# 개발 문서 개요

이 폴더는 프로젝트 개발 문서를 정리합니다.

## 프로젝트 목적
- 이 프로젝트는 하나의 포트폴리오를 만드는 것이 목적입니다.
- 구현 대상은 간단한 쇼핑몰 웹사이트입니다.

## 사용 언어
- 기본 언어는 한국어입니다.

## 폴더 구조
- `.docs/`: 개발 문서
- `.docs/.ask/`: 작업일지(로그)

## 문서 작성 규칙
- 문서는 한국어로 작성합니다.
- 변경/의사결정은 날짜와 함께 기록합니다.
- 기능 구현 전, 요구사항을 먼저 정리합니다.

## 문서 목록
- 요구사항 정의: `requirements.md`
- 기능 목록 및 우선순위: `features.md`
- 화면/흐름 설계: `ui-flow.md`
- 데이터 모델 설계: `data-model.md`
- API 설계: `api.md`
- 테스트 전략: `test-strategy.md`
- 구현 로드맵: `roadmap.md`
