# 구현 로드맵 (현재 상태 반영)

작성일: 2026-02-06

## 1단계: 기반 (완료)
- 프로젝트 초기 설정
- 공통 레이아웃/네비게이션 구성
- 인증(회원가입/로그인/로그아웃, 세션 기반)

## 2단계: 상품 (완료)
- 상품 목록/상세 페이지
- 판매자 상품 등록/수정/삭제
- 판매자 상품 목록

## 3단계: 구매 흐름 (완료)
- 장바구니(담기/삭제/수량 변경)
- 주문 생성
- 주문 내역/상세/완료 페이지

## 4단계: 판매자 주문 (완료)
- 판매자 주문 목록
- 주문 상태 변경

## 5단계: 안정화 (진행 중)
- 권한 검증 보강
- 예외/경계 케이스 테스트 확장
- 문서 정합성 유지

## 6단계: 확장 (예정)
- 상품 검색/필터
- 주문 취소/배송 상세
- 리뷰/문의/위시리스트
