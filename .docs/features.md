# 기능 목록 및 우선순위

우선순위 기준:
- MVP: 로그인 + 상품 조회 + 장바구니 + 주문 + 판매자 주문처리까지 포함
- 다음: MVP 이후 확장
- 보류: 필요 시 추후 검토

## MVP (높음)
1. 회원가입/로그인/로그아웃: 판매자/구매자 구분
2. 상품 목록/검색/필터
3. 상품 상세 보기
4. 장바구니 담기/삭제
5. 장바구니 수량 변경
6. 주문 생성(구매)
7. 주문 내역 조회
8. 판매자 주문 내역 확인
9. 주문 상태 변경(배송 준비/배송 중/배송 완료)
10. 상품 등록

## 다음 (중간)
1. 상품 수정(가격/재고/설명)
2. 상품 삭제
3. 상품 목록/검색(본인 상품 관리용)
4. 재고 관리(수량 변경, 품절 처리)
5. 구매 취소
6. 배송 상태 확인
7. 리뷰 작성
8. 문의 작성(상품 문의)
9. 문의/리뷰 관리(답변/삭제 여부 등, 선택)

## 보류 (낮음)
1. 찜하기(위시리스트)
