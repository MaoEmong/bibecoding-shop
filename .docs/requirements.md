# 요구사항 정의

## 범위
- 간단한 쇼핑몰 웹사이트를 포트폴리오 목적으로 구현합니다.

## 사용자 유형
- 판매자: 상품을 등록/관리하는 사용자
- 구매자: 상품을 탐색/구매하는 사용자

## 인증/계정
- 회원가입 기능이 필요합니다.
- 로그인 기능이 필요합니다.
- 로그아웃 기능이 필요합니다.
- 로그인 시 사용자 유형(판매자/구매자)을 구분합니다.

## 판매자 기능
- 상품 등록
- 상품 수정(가격/재고/설명)
- 상품 삭제
- 상품 목록/검색(본인 상품 관리용)
- 재고 관리(수량 변경, 품절 처리)
- 구매 요청 내역 확인
- 주문 상태 변경(배송 준비/배송 중/배송 완료)
- 배송 처리
- 문의/리뷰 관리(답변/삭제 여부 등, 선택)

## 구매자 기능
- 상품 목록/검색/필터
- 상품 상세 보기
- 장바구니 담기/삭제
- 장바구니 수량 변경
- 상품 구매
- 구매 취소
- 주문 내역 조회
- 배송 상태 확인
- 찜하기(위시리스트)
- 문의 작성(상품 문의)
- 리뷰 작성

## 비고
- 상세 기능(상품, 장바구니, 주문 등)은 추후 정의합니다.
